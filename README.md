# Data-Driven Analysis of Hotel Occupancy and Booking Trends from 2019 to 2026

This repository provides an example use case for the 'Hotel Open Datasets (January 2019 - March 2026)'. The dataset offers comprehensive and up-to-date data on hotel operations, including occupancy rates, room types, pricing, customer demographics, and booking trends in Abu Dhabi. This use case demonstrates how to analyze the data to derive actionable insights.

## Requirements
1. Python 3.7 or higher
2. Libraries:
   - pandas
   - matplotlib

To install the required libraries, use:

bash
pip install pandas matplotlib


## How to Use This Code
1. Download the dataset file (e.g., 'Abu Dhabi Hotels Open Datasets 1 - Supply by zones.xlsx') from the official source.
2. Save the file in your working directory.
3. Update the `file_path` variable in the Python script to the correct path of the downloaded dataset.
4. Run the script to analyze the data and generate a bar chart of the average hotel occupancy rate by region.

## Example Output
The script produces a bar chart that visualizes the average hotel occupancy rate by region over the period from January 2019 to March 2026. The chart is saved as `average_occupancy_rate_by_region.png` in the working directory.

## Applications
- **Tourism Industry Professionals**: Use the insights to design targeted marketing strategies and optimize pricing models.
- **Policymakers and Urban Planners**: Analyze regional trends to inform infrastructure development.
- **Researchers and Analysts**: Study seasonal variations and long-term trends in hotel performance.
